Get started in as little as 5 minutes

Sign Up
Provide a name and contact information and you’re ready to go.
Install the agent
Download the agent and save it to your device. It will call back to saas glue to establish a connection.
Create a script
Upload a script or create one in saas glue. Use any language you want.
Run your script
The agent will pass the script to the appropriate service for execution and return the results to saas glue.
Build a job
Combine scripts in a job with variables, conditional routing and more. Set it to run automatically or trigger it through the API.
